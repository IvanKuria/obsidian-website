# LOTUS

- if $x$ is a discrete random variable and $g$ is a function from $\mathbb{R}$ to$ $\mathbb{R}$, then
$$E(g(X)) = \sum_{x}P(X = x)$$
- where the sum is taken over all possible values of $X$.
