The interval between stimulation and response; the time it takes to complete a single task from start to finish.

## Context

Latency matters whenever responsiveness is important. For example, a network request with 100ms latency takes 100 milliseconds from when the request is sent to when the response arrives. Low latency is critical for interactive applications, while [[bandwidth]] (throughput) matters more for bulk data transfers.
