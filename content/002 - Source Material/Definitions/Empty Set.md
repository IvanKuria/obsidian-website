A set containing no elements, denoted as $\emptyset$ or $\{\}$. The empty set is a subset of every set.

## Context

The empty set is a foundational concept in set theory. For example, the set of all integers that are both even and odd is $\emptyset$. The [[power set]] of the empty set is $\{\emptyset\}$, which has [[cardinality]] 1.
