An arrangement of all of a set's elements in a row; a list without repetition that uses every element of the set. The number of permutations of an $n$-element set is $n!$.

## Context

Permutations are fundamental in combinatorics. For example, the set $\{A, B, C\}$ has $3! = 6$ permutations: ABC, ACB, BAC, BCA, CAB, CBA. Permutations are used in algorithm analysis, cryptography, and probability.
