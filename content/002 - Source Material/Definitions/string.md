A finite sequence of symbols, written without parentheses and commas. Also described as a list of symbols. [[CSE 103]]

## Context

In computer science, strings are one of the most common data types, used to represent text. In formal language theory, strings are finite sequences over an alphabet $\Sigma$, and a [[language]] is a set of strings. The [[empty string]] (length 0) is the simplest string.
