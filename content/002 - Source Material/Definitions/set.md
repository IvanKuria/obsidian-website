A collection of distinct elements (things). Sets are unordered and contain no duplicate elements. Denoted with curly braces, e.g., $A = \{1, 2, 3\}$.

## Context

Sets are the most fundamental object in discrete mathematics. Nearly every mathematical structure (functions, relations, graphs, etc.) is defined in terms of sets. Set operations include [[complement|union, intersection, and complement]], and sets can be described using [[RosterNotation|roster notation]] or [[set builder]] notation.
