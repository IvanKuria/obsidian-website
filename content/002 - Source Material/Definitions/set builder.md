---
title: "Set Builder Notation"
type: definition
domain: discrete-math
aliases: [set builder, set-builder notation]
tags:
  - definition
---

A way to describe a set by specifying a property that its elements must satisfy, used when elements cannot be conveniently listed. Written as $\{x : P(x)\}$ or $\{x \mid P(x)\}$. For example, $\{x \in \mathbb{Z} : x > 0\}$ denotes the positive integers.
