True simultaneous execution of multiple tasks on separate hardware resources. Parallelism equals [[throughput]] ([[bandwidth]]) multiplied by [[latency]].

## Context

Parallelism requires multiple processing units (cores, CPUs, or machines). For example, a quad-core CPU can execute four threads truly in parallel. This differs from [[concurrency]], where threads merely take turns on a single core.
