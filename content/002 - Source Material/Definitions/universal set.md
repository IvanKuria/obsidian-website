---
title: "Universal Set"
type: definition
domain: discrete-math
aliases: [universal sets]
tags:
  - definition
---

The set $U$ that contains all the elements under consideration for a particular discussion or problem. Every set $A$ in the discussion satisfies $A \subseteq U$.
